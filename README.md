# GymTracker - ジムトレーニング記録アプリ

iPhoneで簡単にジムでのトレーニング記録を管理するアプリです。

## 機能

### 📱 ホーム画面
- **天気表示**：東京都豊島区の気温とアイコン表示
- **今日のメニュー**：曜日ごとのトレーニングメニューを自動表示
- **クイックアクション**：トレーニング開始、メニュー管理、過去データ確認

### 🏋️ トレーニング記録
- **自動タイマー**：曜日ごとに異なる時間設定可能（デフォルト70分）
- **セットごと記録**：各セットの回数を個別に入力
- **進捗表示**：前回の同じ曜日との比較で回数が増えた場合は「+」、減った場合は「-」を表示
- **記録保存**：トレーニング終了時に自動保存

### ⚙️ メニュー管理
- **曜日別管理**：7曜日すべてのトレーニング種目を管理
- **簡単編集**：種目の追加・削除・編集が可能
- **フレキシブル**：いつでもメニューを変更可能

### 📊 データ分析
- **進捗グラフ**：曜日ごとのトレーニングボリューム（総回数）を可視化
- **履歴確認**：過去のトレーニング記録を閲覧

### ⏱️ 設定
- **タイマー時間カスタマイズ**：各曜日のトレーニング時間を個別に設定

## 使い方

### 初回セットアップ

1. `setup-menu.html` をブラウザで開く
2. 「メニューを自動登録」ボタンをクリック
3. `index.html` を開く
4. ホーム画面に表示される

### iPhoneへのインストール

1. Safari で `index.html` を開く
2. 下部のシェアボタン → 「ホーム画面に追加」
3. ホーム画面からアプリを起動

## 使用方法

### トレーニング開始
1. ホーム画面の「トレーニング開始」ボタンをタップ
2. 各種目のセットごとに回数を入力
3. 「開始」でタイマースタート
4. トレーニング終了後「トレーニング終了」をタップ

### メニュー管理
1. ホーム画面の「メニュー管理」をタップ
2. タブで曜日を選択
3. 種目を編集または「種目を追加」で新しい種目を追加
4. 自動保存される

### 設定変更
1. ホーム画面の⚙️をタップ
2. 各曜日のトレーニング時間を入力
3. 自動保存される

## データ保存

- すべてのデータはiPhoneのローカルストレージに保存されます
- クラウド同期はありません
- iPhoneのブラウザキャッシュを削除するとデータが失われます

## 技術仕様

- **フロントエンド**：HTML5 + CSS3 + JavaScript
- **ストレージ**：LocalStorage
- **天気API**：Open-Meteo（無料）
- **グラフ表示**：Chart.js
- **PWA対応**：オフラインで動作可能

## トラブルシューティング

### データが表示されない
- キャッシュをクリアして再度開く
- `setup-menu.html` でメニューを再登録

### 天気が表示されない
- インターネット接続を確認
- Open-Meteo APIが利用可能か確認

### タイマーが音を出さない
- ブラウザの音声設定を確認
- 通知許可を確認
